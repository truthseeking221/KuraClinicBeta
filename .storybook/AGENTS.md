# Storybook Configuration Guidance

- Read and follow [`../.codex/skills/design-kura-ui/SKILL.md`](../.codex/skills/design-kura-ui/SKILL.md) before changing Storybook configuration, decorators, themes, viewports, rendering behavior, or visual-test setup.
- Read and follow [the ReUI × BoardUI × Kura component build guide](../.codex/skills/design-kura-ui/references/reui-boardui-kura-component-build-guide.md) when configuration affects component intake, story metadata, tests, viewports, screenshots, ownership validation, or the definition of done.
- Preserve Storybook as the canonical rendering and verification environment for Kura UI.
- Ensure configuration supports responsive inspection, accessibility, realistic states, stable token and font loading, and direct rendered verification.
- Keep all agent-authored instructions, documentation, comments, and Storybook-facing copy in English unless the user explicitly requests another language.
