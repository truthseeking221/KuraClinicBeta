# Identity provider brand assets

These assets are exposed only through the canonical Kura provider-mark wrappers in
`src/components/ui/icons/provider-marks.tsx`.

## Google

- `google-g.png` is Google's standard-colour gradient Super G.
- Source: https://developers.google.com/static/identity/images/g-logo.png
- Branding guidance: https://developers.google.com/identity/branding-guidelines
- Retrieved: 2026-07-17

Do not recolour, redraw, or replace this asset with a monochrome interface icon.
